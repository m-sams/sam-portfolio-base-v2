Target Shooter

This target shooting game marks some of my beginner work using Unreal Engine. Shooting all five targets in time prompts the user to a win screen with the option to restart, and if the user loses they're taken to a losing screen in which they also have the option to restart. This demo game touches on the basics of game development using UE including scene building and character interaction scripting.

Software & Programs: Unreal Engine v5

Completed: Spring 2022

ProgPros

ProgPros is a conceptual application with web and mobile versions that targets users who are fans of progressive metal and guitarists.

The design is informed by taking a particular passion of mine and following basic web design principles to produce a high-fidelity wireframe to showcase how I would implement a website of this nature. In the FIGMA file, you can click through light and dark modes of the page on different devices along with an additional cards page to show ideal hovering functionality when the information is hidden.

Software & Programs: FIGMA

Completed: Spring 2022

100 Suits Student Organization (1SSI)

1SSI is a media campaign modeled around 100 Suits for 100 Men, a non-profit organization that provides formal clothing to the underpriveledged, prepares food giveaways, and actively participates in community violence prevention. This campaign targeted Borough of Manhattan Community College (BMCC) students who have an interest in socially-minded work to get them to work with the 100 Suits for 100 Men Organization.

I was responsible for a website showcasing relevant information about 1SSI, 100 Suits for 100 Men, and the other relevant aspects of the campaign.

Software & Programs: HTML, CSS, JS, Visual Studio Code, Github Deployment

Last Updated: Fall 2022 (WIP)
